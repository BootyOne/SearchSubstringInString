Задача: Алгоритмы поиска подстроки в строке

Автор: Романенко Михаил ФТ-203

Краткое описание: Сравнение производительности алгоритмов поиска подстроки в строке по памяти и времени.
Для теста вам достаточно запустить файл "test.py" через консоль путём ввода в неё в папке с проектом:"python test.py"
И спустя какое-то время(Зависит от мощности вашего компьютера), когда программа закончит своё выполнение,
результат можно будет увидеть в текстовом файле "test.txt"

Состав проекта:

Algorithms.py - файл с реализацией алгоритмов
(Алгоритм грубой силы, алгоритм Кнута-Мориса-Пратта(стандартный и продвинутый), алгоритм Рабина-Карпа, Алгоритм Бойера-Мура)
Данные алгоритмы были выбраны из-за их популярности и большого количества информации о них в интернете.
Так как больший интерес вызывает проверка наиболее используемых алгоритмов.

Main.py - файл с проведением тестов алгоритмов

benchmark.py - реализация функции для тестирования алгоритмов

alphabet.py - реализация данных и искомых строк

tests.py - набор тестов на проверку корректности работы алгоритмов

Более подробно про особенности алгоритмов и их тестирование смотрите в отчёте.
